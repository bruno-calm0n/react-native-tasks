# React Native Tasks

Um aplicativo simples para organizar tarefas diarias de forma rapida, limpa e intuitiva.

## Sobre

O projeto permite criar uma lista inicial de tarefas e adicionar novos itens pela tela principal.  
Foi criado com Expo, React Native e TypeScript, seguindo uma estrutura componentizada e sem bibliotecas externas de UI.

## Como rodar localmente

Instale as dependencias:

```bash
npm install
```

Inicie o Expo:

```bash
npm run start
```

Ou:

```bash
npx expo start
```

Depois, escaneie o QR Code com o Expo Go.
